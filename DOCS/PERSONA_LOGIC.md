
# Persona Modeling Logic

## 1. 社會階層與經濟模型 (Socio-Economic Model)
基於 `data/taiwan_sociology.ts`。

### 1.1 收入階級 (Income Classes)
- **Survival**: < P10。生存線。
- **Tight**: P10 - P25。拮据線 (Trade-off)。
- **Stable**: Median。小康穩定。
- **Affluent**: P90。富裕自由。
- **Elite**: > P99。頂層資產階級。

## 2. 社會動力學 (Social Dynamics)
- **Money Type**: Blood_Sweat (血汗錢) vs Easy_Windfall (資產/快錢)。
- **Coping Strategy**: 
  - **Mom_Bank**: 靠爸族 (高消費低收入)。
  - **Eat_Cheap_Wear_Rich**: 精緻窮 (高面子低底薪)。
  - **Stealth_Wealth**: 隱形富豪 (低面子高資產)。

## 3. Prism 9.0 戰略矩陣 (Strategic Coordinates)
用於 Product Mirror 模式，計算市場切入點。

### 3.1 痛點張力 (Demand Tension) - X軸
- **定義**: 問題發生的頻率 x 不解決的代價。
- **高分 (80-100)**: 每日發生、社交死亡、金錢損失。
- **低分 (0-40)**: Nice to have。

### 3.2 競品鎖定 (Competitive Lock-in) - Y軸
- **定義**: 轉換成本與既有習慣的強度。
- **高分**: 生態系鎖定、高轉換成本。
- **低分**: **Poison Protocol 生效** (競品有重大災情或信任破產)。

### 3.3 切入可行性 (Entry Feasibility) - Z軸
- **定義**: 該 Persona 接受新方案的門檻 (知識/技術/通路)。
- **高分**: 低學習門檻、低試錯成本。

## 4. 現實校準 (Reality Check & Anchoring)
- **Reality Lock**: 當 `Insolvent` 或 `Delusional` 狀態觸發時，轉換分數 (Conversion Score) 的阻力係數強制 x 2.5。
- **Affordability Ratio**: 產品月付額 / 可支配所得。若 > 30% 則產生指數級阻力。

## 5. 意圖稜鏡 (Search Psychology)
- **Urgent Pain**: 搜尋「怎麼辦」、「急救」。點擊「3分鐘解決」的標題。
- **Rational**: 搜尋「比較」、「規格」。點擊「深度評測」的標題。
- **Skeptic**: 搜尋「缺點」、「災情」。只點擊「負面揭露」的標題。

## 6. 時空戰略 (Chronos)
- **Liquidity Cycle**: 發薪日 (月初) vs 生存日 (月底)。
- **Bandwidth**: 事件驅動的心智頻寬 (Occupied vs Open)。
